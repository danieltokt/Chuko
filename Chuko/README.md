# 🎯 Chuko Classic

A digital version of **Chuko** — a traditional Kyrgyz game, built with Next.js and physics-based gameplay inspired by Angry Birds.

> ⚠️ This is an early version (MVP). More features coming soon.

## 🎮 About the Game

Chuko is a traditional game played for centuries in Kyrgyzstan. Players throw small figurines to knock down targets. This game brings that spirit to the browser with real physics.

- Pull the slingshot, aim, and launch your Chuko
- Knock down all white bone-shaped targets to win
- Use fewer Chukos to earn more stars ⭐⭐⭐
- 15 levels of increasing difficulty

## 🚀 Tech Stack

- **Next.js 15** — React framework
- **Matter.js** — 2D physics engine
- **Tailwind CSS** — styling
- **TypeScript** — type safety
- **shadcn/ui** — UI components

## 🛠 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Build

```bash
npm run build
npm start
```

## 🌐 Deploy

Deployed on [Vercel](https://vercel.com). Any push to `main` triggers a new deployment.
